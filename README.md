# Diseños_html-css
## Descripción de los diseños realizados con html+css
Consiste en ejercicios practicos realizados en html y css, para poder demostrar los conocimientos adquiridos en el curso visto, cabe recalcar que solo es la parte visual, no se realizo nada de backend
Los diseños realizados son los siguientes:
+ Tarjeta de presentación
+ Tarjeta de Viajes
+ Web ONG

### Tarjeta de presentación
El objetivo de esta práctica era crear una tarjeta de presentación personal utilizando únicamente HTML y CSS, aplicando los conceptos vistos hasta ahora en el curso.
